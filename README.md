# Fee Calculator
Calculate fees for various websites that resellers use.

![Screenshot](https://i.imgur.com/dDrt0t9.png)

## Setup

Requirements
- <a href="http://nodejs.org/">Node.js</a>
- Internet

Development Setup:

```sh
> git clone https://github.com/SunstroUS/FeeCalculator.git
> npm install
> npm run start
```
